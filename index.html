mpyOS for MicroPython documentation

Welcome! This is the documentation for mpyOS for MicroPython v1.0.10, last updated 20 Jul 2025.

mpyOS is a console operating system for Micropython. The following distributions of Micropython are supported.- also known as ports - that MicroPython runs on.
