debug¶

OS Debug level¶

  1. debug - is Standard debug (Userlevel).

  2. debug 1 is equivalent to True.

  3. debug 2 os debug decoding.

APP Debug level¶

  1. debug 11 app debug with globals.

  2. debug 12 app debug with locals.

  3. debug 13 app debug with globals and locals.
