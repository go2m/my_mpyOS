Willkommen Darstellung von Beispiel Texten mit Sphinx!
======================================================

**Sphinx** (/lu'make/) Sphinx ist ein Software-Dokumentationswerkzeug,
welches reStructuredText in HTML-Seiten sowie viele weitere Formate wie PDF, EPUB und Manpages umwandelt.
It pulls data from the `Open Food
Facts database <https://www.sphinx-doc.org/en/master/usage/restructuredtext/index.html>`_ and offers a *simple* and
*intuitive* API.

.. note::

   This project is under active development.
